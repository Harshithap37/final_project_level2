What is Formal Verification?

Formal verification uses mathematical proofs to check the correctness of software/hardware systems.

  • Model checking (state-space exploration)
  • Theorem proving (interactive & automated)
  • Abstract interpretation & static analysis

Logic Foundations

HOL, dependent types, temporal logics (LTL/CTL), Hoare logic, refinement.

Suggested Path

  1. Pick a proof assistant (Isabelle/HOL, Coq, Lean)
  2. Finish a small case study end-to-end
  3. Add SMT/model checker integrations
